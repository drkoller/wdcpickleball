Instructions for removing computer from Pickleball Cannon

  1. Remove the ball machine cover.
  2. (Video) With the cover off you have access to the connectors. There is a tab that you squeeze on the connector to remove them. As you take them off, label them with tape so you know where to put them back.
  3. (Video) More connector removal.
  4. (Video) Take a photo of this connector before you remove it so that you know its orientation to reinstall.
  5. (Video) There is a single nut on the bottom of the control panel bracket that needs to come off. It is a bit awkward. Be careful not to damage anything when removing it. If you have an extension, socket and knuckle it makes it easier.
  6. (Video) The computer will now lift off easily.
  7. (Video) When we return the board it will have a new power switch as demonstrated.
  8. (Video) We will return the computer with a template. You will need to drill the cover so that you acess to the new switch. Do you have access to drill bits, likely around 3/4" to 1"?
